# プロジェクトルール

## 技術スタック 

- 言語: Rust
- 競技用コマンド: cargo-compete
- スニペット作成: cargo-snippet

## コーディング規約

- 変数の命名はその場で適切な名前を使用する
- 標準ライブラリと`compete.toml`に記載されている**dependencies**の一覧のクレートのみを使用する
- 構造体やメソッド定義する場合はどのような振る舞いをするのかのコメントを1,2行で記載する

## 禁止事項

- 許可なく勝手に実装しない
- dependencies以外のクレートを使用しない
- 設定ファイルは許可なく修正しない
- 変数名を適当にaとかbにしない

## テスト

- コンテストの問題をテスト: cargo compete test
- 作成したモジュールのテスト: cargo test
